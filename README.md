## 项目简介

AI 周报助手是一款智能周报生成工具，根据工作要点自动生成专业周报和下周计划。

## 核心功能

- 📝 **智能周报**：输入工作要点，AI 自动扩展生成专业周报
- 📅 **下周计划**：根据本周工作自动生成下周计划
- 🎯 **多种模板**：技术研发、产品运营、销售商务、行政人事
- 📋 **一键复制**：快速复制到其他平台
- 💾 **历史管理**：保存和管理历史周报

## 技术栈

- HTML/CSS/JavaScript
- Tailwind CSS
- OpenAI API
- 阿里云 ESA EdgeKV

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
